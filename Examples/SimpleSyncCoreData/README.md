# SimpleSync (Core Data)

A minimal Core Data app demonstrating Ensembles sync between two simulated devices.

Both panels share a local cloud directory so you can see sync working without multiple physical devices or CloudKit setup. Tap "Randomize" on one side and watch the number sync to the other.

## Requirements

- iOS 16+
- Xcode 16+

## Setup

Open `SimpleSync.xcodeproj` in Xcode. The project references the Ensembles3 package from GitHub — Xcode will resolve it automatically.
